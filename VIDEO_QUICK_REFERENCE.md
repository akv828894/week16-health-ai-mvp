## 🚨 IF SOMETHING GOES WRONG

**Server crashed?**
- Restart backend: `npm run dev` in server folder
- Restart frontend: `npm run dev` in client folder

**Can't register?**
- Check MongoDB connection
- Use different email address

**AI not responding?**
- Check GEMINI_API_KEY in .env
- Has a placeholder, need real key

**Mobile view won't open?**
- Refresh page (Ctrl+R)
- Try Firefox if Chrome issues
